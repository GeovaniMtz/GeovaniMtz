## 🚀 Proyectos Destacados

Aquí hay algunos de los proyectos en los que he trabajado:

### 1. OGPS Reports (Proyecto Full-Stack)
* **Descripción:** Sistema de gestión y reporte de incidentes, desarrollado como proyecto final para Ingeniería de Software.
* **Tecnologías Clave:**
    * **Backend:** API REST desarrollada con **Kotlin** y **Spring Boot**.
    * **Frontend:** Interfaz de usuario construida con **JavaScript** (React).
    * **Base de Datos:** **PostgreSQL**.
    * **Herramientas:** Pruebas con **Postman** y autenticación con **JWT**.
* **Repositorios:**
    * [Ver Código del Backend](https://github.com/ingenieria-software-7009-2025-2/OGPS-REPORTS)
    * [Ver Código del Frontend](https://github.com/ingenieria-software-7009-2025-2/ogps-reports-frontend)

### 2. SyrionPC-s (Integración de APIs)
* **Descripción:** Práctica de backend para una tienda ficticia (SyrionPC), enfocada en la integración y consumo de múltiples servicios y APIs externas.
* **Tecnologías Clave:** **Python**, **Spring Boot**.
* **Base de Datos:** **MySQL**.
* **Herramientas:** Pruebas de API con **Postman** y autenticación con **JWT**.
* **Repositorio:**
    * [Ver Código del Repositorio](https://github.com/GeovaniMtz/SyrionPC-s)
